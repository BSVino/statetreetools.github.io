# Gameplay Tags

Conditions for checking gameplay tag state on an actor's Ability System Component.

| Condition | Description |
|-----------|-------------|
| [Has Matching Gameplay Tag](/conditions/has-matching-gameplay-tag) | True if the actor's ASC has the specified tag |
| [Has All Matching Gameplay Tags](/conditions/has-all-matching-gameplay-tags) | True if the actor's ASC has every tag in the container |
| [Has Any Matching Gameplay Tags](/conditions/has-any-matching-gameplay-tags) | True if the actor's ASC has at least one tag from the container |
| [Matches Gameplay Tag Query](/conditions/matches-gameplay-tag-query) | True if the actor's ASC satisfies a gameplay tag query |
