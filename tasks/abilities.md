# Abilities

Tasks and effects from the StateTreeToolsGameplayAbilitySystem plugin.

| Task | Description |
|------|-------------|
| [Try Activate Ability By Class](/tasks/try-activate-ability-by-class) | Activates a gameplay ability on an actor's ASC and waits until the ability ends. |
| [Send Gameplay Event To Actor](/tasks/send-gameplay-event-to-actor) | Sends a gameplay event to an actor's ASC on enter, exit, or both. |
| [Apply Gameplay Effect to Self](/tasks/apply-gameplay-effect-to-self) | Applies a gameplay effect to the bound actor's own ASC. |
| [Apply Gameplay Effect to Target](/tasks/apply-gameplay-effect-to-target) | Applies a gameplay effect from one actor's ASC to a separate target actor's ASC. |
| [Wait For Gameplay Tag Added](/tasks/wait-for-gameplay-tag-added) | Waits until a gameplay tag is added to an actor's ASC. |
| [Wait For Gameplay Tag Removed](/tasks/wait-for-gameplay-tag-removed) | Waits until a gameplay tag is removed from an actor's ASC. |
