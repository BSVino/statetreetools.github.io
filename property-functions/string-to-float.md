# String To Float

**Plugin:** StateTreeToolsCore
**Category:** Utilities \| String

Parses a string as a float. If the string does not represent a valid number the result is `0`. Use this to decode a numeric value that arrived as a string — for example from a gameplay tag payload or a name-keyed data table.

---

## Configuration

### Value
The string to parse as a float.

### Result
The float value parsed from **Value**, or `0` if parsing fails.
