# Utilities

Property functions for string conversion and general value utilities. **Requires UE 5.5+.**

| Property Function | Description |
|-------------------|-------------|
| [Get Display Name](/property-functions/get-display-name) | Get an actor's display name as a string |
| [Boolean To String](/property-functions/boolean-to-string) | Convert a bool to "true" or "false" |
| [Float To String](/property-functions/float-to-string) | Convert a float to its string representation |
| [Integer To String](/property-functions/integer-to-string) | Convert an integer to its string representation |
| [Name To String](/property-functions/name-to-string) | Convert an FName to a string |
| [Text To String](/property-functions/text-to-string) | Convert an FText to a plain string |
| [String To Float](/property-functions/string-to-float) | Parse a string as a float |
| [String To Integer](/property-functions/string-to-integer) | Parse a string as an integer |
| [String To Name](/property-functions/string-to-name) | Convert a string to an FName |
| [String To Text](/property-functions/string-to-text) | Wrap a string in an FText |
