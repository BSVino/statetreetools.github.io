# Get Enhanced Input World Subsystem

**Plugin:** StateTreeToolsCore
**Category:** Enhanced Input
**Availability:** StateTree Tools **2.0+**

Returns the `UEnhancedInputWorldSubsystem` for the current world. Takes no inputs — the world is resolved automatically from the StateTree execution context.

## Unreal Engine Version Notes

Property functions are not available in UE 5.4. This node requires **UE 5.5 or later**.

---

## Configuration

### Output
The `UEnhancedInputWorldSubsystem` for the current world, or None if no world is available.
