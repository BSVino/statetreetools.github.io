# Gameplay Attributes

Property functions for reading Gameplay Ability System (GAS) float attributes from an actor's Ability System Component.

| Property Function | Description |
|-------------------|-------------|

| [Get Float Attribute Base](/property-functions/get-float-attribute-base) | Read the base (unmodified) value of a float attribute |
