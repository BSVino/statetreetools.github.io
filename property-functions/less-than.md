# <

**Plugin:** StateTreeToolsCore
**Category:** Math \| Float

Returns `true` if **Left** is strictly less than **Right**. Use this to drive a boolean binding that can then be tested in a transition condition or used as input to another property function.

---

## Configuration

### Left
The value on the left-hand side of the comparison.

### Right
The value on the right-hand side of the comparison.

### Result
`true` if **Left** < **Right**, otherwise `false`.
