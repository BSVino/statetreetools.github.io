# Tan (Degrees)

**Plugin:** StateTreeToolsCore
**Category:** Math \| Float

Returns the tangent of an angle given in degrees. Use this when your angle is already in degrees. The result is unbounded and approaches infinity as the angle approaches ±90°.

---

## Configuration

### Value
The angle in degrees.

### Result
The tangent of **Value**.
