# Get PI

**Plugin:** StateTreeToolsCore
**Category:** Math \| Float

Returns the mathematical constant π (approximately `3.14159265`). Use this as a bindable source when you need π in a calculation chain — for example as input to a multiplication or a trigonometric conversion.

---

## Configuration

### Result
The value of π as a float.
